How the shortening actually works

This tool talks to TinyURL's public create endpoint to generate a real short link, with a same-device fallback if that request is ever blocked by the browser. Services like Bitly require a server-side API key and can't be called securely from a page like this one without a backend — so this build is structured to swap in a Bitly-backed endpoint later with one function changed in app.js, without touching the UI.
