# Shortn

**Shortn** is a clean, efficient web-based URL shortener designed to transform long, complex URLs into concise, shareable links.

## 🚀 Key Features
*   **Simple Input:** Intuitive interface to quickly input long URLs.
*   **Instant Shortening:** Generates compact links in real-time.
*   **Responsive Design:** Fully optimized for both desktop and mobile users.
*   **Clean UI:** Minimalist design focused on speed and ease of use.

## 🛠 Tech Stack
*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Deployment:** GitHub Pages
